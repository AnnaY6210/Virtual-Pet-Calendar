Help

Usage

  • Tasks are added via Google Calendar
  • Only the 40 most recent tasks are displayed
  • Click the "Claim Tasks" button to recieve rewards for all tasks completed on-time after the "Last Claimed" date

Claiming

  • Only tasks completed with 1 day of the due date will reward currency
  • Repeating tasks must be claimed before their next instance, as the old one is deleted automatically
  • Upon claiming rewards, additional tasks completed on the current day cannot be claimed

Setting Rewards

  • The default reward per task is 5
  • Rewards can be set on a per Task List basis with vertical bar "|" and a positive whole number.

    For example: Important | 200

    will reward 200 for every task in the "Important" task list
